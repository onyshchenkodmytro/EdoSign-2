У ході другої лабораторної роботи було реалізовано процес розгортання та використання власного NuGet-пакета у багатосередовищному середовищі на базі Vagrant.

Спочатку було створено бібліотеку EdoSign.Signing, яка містить інтерфейс ISigner та клас RsaSigner, що реалізує підписання документів за допомогою RSA. Після цього бібліотеку було спаковано у NuGet-пакет і розгорнуто на приватному сервері BaGet, запущеному у віртуальній машині Ubuntu через Docker.

Далі у другій віртуальній машині (ubuntu-client) було налаштовано середовище .NET, оновлено файл NuGet.Config для підключення до приватного BaGet-сервера, а потім виконано завантаження пакета та інтеграцію його у проєкт EdoSign.Api. Код, що відповідав за підпис документів, був замінений на використання підключеної бібліотеки з пакета через механізм Dependency Injection.

Після успішної збірки та запуску проєкту було перевірено роботу API — інтерфейс Swagger став доступним, що підтвердило коректність підключення пакета, налаштування сервера та працездатність системи.
